Mastery at points

Example

Weighted Average: Most recent result gets 65% weight, average of all other results gets 35% weight. Emphasizes recent performance while considering historical data.

Example: Student Scores: [2.0, 3.0, 4.0]

Calculation: Latest (4.0) × 0.65 + Average of others (2.5) × 0.35

Final Score: 3.5 (4.0 × 0.65 + 2.5 × 0.35)

Decaying Average: Exponential decay where each older assessment gets progressively less weight. Recent performance has exponentially more influence.

Example: Student Scores: [2.0, 3.0, 4.0] with 65% decay factor

Calculation: 4.0×1.0 + 3.0×0.65 + 2.0×0.65²

Final Score: 3.6 (4.0 + 1.95 + 0.845 ÷ 2.795)

n Number of Times: Student must achieve mastery (≥3.0 points) at least 5 times. Only scores meeting mastery are counted in the average.

Example: Student Scores: [2.0, 3.5, 2.5, 4.0, 3.5, 3.2, 2.8] (Mastery = 3.0)

Mastery scores: [3.5, 4.0, 3.5, 3.2] = 4 times ≥ 3.0

Result: Not Achieved (need 5 times, only achieved 4)

Most Recent Score: Only the latest assessment result counts. Reflects current performance level.

Example: Student Scores: [2.0, 3.0, 4.0]

Final Score: 4.0 (most recent)

Highest Score: Best performance achieved across all assessments. Shows peak capability.

Example: Student Scores: [2.0, 4.0, 3.0, 2.5]

Final Score: 4.0 (highest achieved)

Simple Average: Sum of all scores divided by count. Equal weight to all attempts.

Warning: May penalize students with early poor performance. Not recommended for mastery-based learning.

Example: Student Scores: [2.0, 3.0, 4.0]

Final Score: 3.0 ((2.0 + 3.0 + 4.0) ÷ 3)

No Point (Binary): Use this when you are using criteria against Met or Not Met/Refer

We recommend using this measure when there are no specific grade descriptors (e.g. Pass, Merit or Distinction)
