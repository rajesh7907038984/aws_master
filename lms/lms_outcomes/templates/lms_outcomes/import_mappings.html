Rubric-Outcome Mapping Format Documentation

Rubric-outcome mappings connect specific rubric criteria to learning outcomes, enabling automatic outcome calculation based on rubric evaluations.

CSV Format

Your CSV file must contain the following columns:

  • rubric_title - The exact title of the rubric
  • criterion_description - The description of the rubric criterion (partial match allowed)
  • outcome_title - The exact title of the outcome
  • weight - Weight of this criterion in outcome calculation (0.0-1.0)

Sample CSV Content

rubric_title,criterion_description,outcome_title,weight
"Assignment Rubric","Critical Thinking","Analyze Complex Problems",1.0
"Assignment Rubric","Written Communication","Communicate Effectively",0.9
"Quiz Rubric","Problem Solving","Solve Mathematical Problems",0.8
"Discussion Rubric","Collaboration","Work Effectively in Teams",1.0

Important Notes

  • Rubric titles and outcome titles must match exactly (case-sensitive)
  • Criterion descriptions allow partial matching for flexibility
  • Weights must be between 0.0 and 1.0
  • If multiple criteria with the same description exist, the system will try exact matching first
  • Existing mappings with the same criterion-outcome pair will be updated with new weights
  • You can only import mappings for rubrics and outcomes within your branch/business access

Outcome Calculation Methods

Once mappings are created, outcomes will be automatically calculated using the method specified for each outcome:

  • Weighted Average: Recent evaluations receive more weight
  • Decaying Average: Exponential decay favoring recent scores
  • n Number of Times: Requires mastery achieved n times
  • Most Recent: Uses only the latest evaluation
  • Highest Score: Uses the best performance
  • Simple Average: Equal weight to all evaluations
  • No Point: Binary met/not met assessment

After Import

After importing mappings, outcome evaluations will be automatically calculated for all students who have existing rubric evaluations. You can also use the Recalculate Evaluations feature to manually trigger recalculation if needed.
