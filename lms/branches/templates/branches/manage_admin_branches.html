How Branch Assignments Work

  • Admin users can be assigned to multiple branches within the same business
  • Each admin has a primary branch (their main assignment) and can have additional branches
  • Admin users can switch between their assigned branches using the branch switcher in the header
  • When switching branches, the admin sees only data from the selected branch
  • Super admins can only assign branches within businesses they have access to
